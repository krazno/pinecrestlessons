Functions: little machines.

A function is a reusable chunk of code. You give it some input, it does work, it hands you back an output. Like a vending machine, but for math.

# the world's simplest function
def square(n):
    return n * n

# now use it as many times as you want
print(square(5))   # → 25
print(square(12))  # → 144

Now let's give a function to a turtle and watch geometry happen for free:

import turtle
t = turtle.Turtle()

def polygon(sides, size):
    for i in range(sides):
        t.forward(size)
        t.left(360 / sides)

polygon(6, 100)   # hexagon
polygon(100, 8)   # near-perfect circle 🤯
