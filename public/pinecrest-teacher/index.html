1. Give It a Role

Tell the AI what kind of expert or teacher lens to use.

Weak Prompt

Make a review activity.

Better Prompt

Act as an experienced teacher. Create a 10-minute review activity for students who need practice with the main skill from today’s lesson.

Try It: Act as a ________.

Works for any subject

2. Show an Example

Show the AI what good looks like before asking for more.

Weak Prompt

Write feedback comments.

Better Prompt

Here is one example of my feedback style: “You made a clear point. Your next step is to explain your evidence more fully.” Now write 5 comments in this same style.

Try It: Add one example you already use.

Works for any subject

3. Ask for Step-by-Step Thinking

Ask the AI to identify criteria or skills before giving the final answer.

Weak Prompt

Grade this response.

Better Prompt

First identify the main skill being assessed. Then give one strength, one area to improve, and one next step.

Try It: First identify the criteria. Then complete the task.

Works for any subject

4. Ask It to Ask You Questions

Have the AI ask you questions before it builds the final product.

Weak Prompt

Make me a lesson.

Better Prompt

Before creating the lesson, ask me 3 questions about my students, the topic, and the goal.

Try It: Ask me 3 questions first.

Works for any subject

5. Roleplay a Scenario

Turn the AI into a practice partner — student, parent, or confused learner.

Weak Prompt

Help students practice.

Better Prompt

Roleplay as a student who is confused. Ask me one question at a time so I can practice explaining the concept clearly.

Try It: Roleplay as ________.

Works for any subject

Full Prompt Glow-Up

Combine all five moves in one prompt.

Weak Prompt

Make a review activity for my students.

Better Prompt

Act as an experienced teacher. Before answering, ask me 3 questions about my class, topic, and goal. Then create a 10-minute review activity. Use this format: 1. Warm-up 2. Partner task 3. Exit ticket First explain what skill the activity builds. Then roleplay as a student who might be confused and ask me one question.

Build One Prompt

Fill in the blanks and paste into your LLM.

Act as a ________. I teach ________. My students are working on ________. Before answering, ask me 3 questions. Then create ________. Use this format: ________. Afterward, roleplay as ________.
