Who can tell me what's a parameter and what's an argument?

parameter_and_argument.py
1def my_function(name): # name is a parameter
2 print("Hello", name)
3
4my_function("Emil") # "Emil" is an argument

Parameter = the variable in the function definition (name). Argument = the actual value you pass when you call the function ("Emil").

What is a positional argument?

When you call a function without naming each value, the arguments are matched by position — first value → first parameter, second value → second parameter. Order matters.

positional_args.py
1def my_function(animal, name):
2 print("I have a", animal)
3 print("My", animal + "'s name is", name)
4
5my_function("dog", "Buddy")

"dog" goes to animal and "Buddy" goes to name because of the order.

Sending a list as an argument

You can pass a whole list into a function as one argument. Inside the function, use a for loop to work with each item.

list_argument.py
1def my_function(fruits):
2 for fruit in fruits:
3 print(fruit)
4
5my_fruits = ["apple", "banana", "cherry"]
6my_function(my_fruits)

The list my_fruits is passed as one argument. The loop prints each fruit on its own line.

Python’s turtle module begins with a turtle at (0, 0) in the x/y plane. forward() moves the turtle in the direction it is facing, right() rotates it clockwise, and turtle graphics gives learners immediate visual feedback — useful for teaching programming concepts.
