Initials on a Grid

Difficulty: middle · core academic task Time: 10–15 minutes

Main goal: Use coordinates and decomposition to create one or two initials drawn with turtle using line segments and grid points.

Your directions

  1. Step 1: Choose one initial.
  2. Step 2: Sketch it on the 4 by 6 grid (paper or use the interactive grid below).
  3. Step 3: Write each line segment as a start point and end point.
  4. Step 4: Code one segment.
  5. Step 5: Run and check.
  6. Step 6: Add the next segment.
  7. Step 7: Add your second initial if time allows.

Minimum: at least 2 colors, 2 comments, one reusable function (e.g. draw_grid_line()), test one segment at a time, screenshot + code turn-in below.

Grid explanation

The x-value moves left and right. The y-value moves up and down.

Each line needs a start point and an end point.

Example — letter H

  • Left line: (0, 0) to (0, 6)
  • Right line: (4, 0) to (4, 6)
  • Middle line: (0, 3) to (4, 3)
draw_grid_line(0, 0, 0, 6, color="blue")
draw_grid_line(4, 0, 4, 6, color="blue")
draw_grid_line(0, 3, 4, 3, color="blue")

Teacher example · code

import turtle

SCALE = 40  # pixels per grid unit
OFFSET_X = -80
OFFSET_Y = -120

def to_screen(gx, gy):
    return OFFSET_X + gx * SCALE, OFFSET_Y + gy * SCALE

def draw_grid_line(x1, y1, x2, y2, color="white"):
    t.penup()
    t.goto(*to_screen(x1, y1))
    t.pendown()
    t.color(color)
    t.goto(*to_screen(x2, y2))

t = turtle.Turtle()
t.speed(0)
screen = turtle.Screen()
screen.bgcolor("black")

# Letter H on the 4x6 grid
draw_grid_line(0, 0, 0, 6, color="cyan")
draw_grid_line(4, 0, 4, 6, color="cyan")
draw_grid_line(0, 3, 4, 3, color="magenta")

turtle.done()
