Python

What is Python?

Python is a programming language — a way to give a computer clear, step-by-step instructions. You write Python in a file, run it, and the computer follows what you wrote.

How does Python run your program?

Python reads your file from top to bottom and runs each line in order. That is why you write clear steps, one after another.

Functions

What is a function, and what does it do?

A function is a named set of steps in your program. It lets you reuse work: you define it once, then run it whenever you need the same job done — without copying and pasting code.

What keywords go with functions in Python?

Key words: def (start a function), the function name, parameters in parentheses, the indented body, and a call with () to run it. return sends a value back to the line that called the function.

What is a function call?

A call is the line that runs the function — you write the function name with parentheses, like send_out_pokemon("Pikachu"). The computer jumps to the function body, does those steps, then continues.

Turtle graphics

What is the use of pen-up and pen-down?

penup() lifts the pen so the turtle can move without drawing a line — useful for jumping to a new spot. pendown() puts the pen back down so the turtle draws again when it moves.

What are some methods and functions available in Turtle?

Examples: forward and backward to move; left and right to turn; penup and pendown; goto to jump to coordinates; color and width for the line; shape for the icon; plus import turtle, turtle.Turtle(), and turtle.done() to start and finish.

Why code with Turtle?

Because the turtle is like coding a robot. It is a small “robot” on the screen: you give it commands (move, turn, lift the pen), and it carries them out — the same idea as programming a real device step by step.

What happens when you pass three values in order? Order matters The 1st value goes to the 1st input, the 2nd to the 2nd, and so on.

That is computer science in real life: use code, machines, and smart design to solve real problems.
