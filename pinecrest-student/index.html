fn functions_in_python.py parameters · positional · lists

Example

example.py
1def my_function(name): # name is a parameter
2 print("Hello", name)
3
4my_function("Emil") # "Emil" is an argument

Parameter = the variable in the function definition (name). Argument = the actual value you pass when you call the function ("Emil").

The phrase Keyword Arguments is often shortened to kwargs in Python documentation.

Positional Arguments

When you call a function with arguments without using keywords, they are called positional arguments.

Positional arguments must be in the correct order:

Example

positional_args.py
1def my_function(animal, name):
2 print("I have a", animal)
3 print("My", animal + "'s name is", name)
4
5my_function("dog", "Buddy")

"dog" goes to animal and "Buddy" goes to name because of the order — first value → first parameter, second value → second parameter.

Sending a List as an Argument

You can pass a whole list into a function. The function can loop through each item:

Example

list_argument.py
1def my_function(fruits):
2 for fruit in fruits:
3 print(fruit)
4
5my_fruits = ["apple", "banana", "cherry"]
6my_function(my_fruits)

The list my_fruits is passed as one argument. Inside the function, the for loop prints each fruit on its own line.
