🎯 Your mission

You're going to create a simple colorful turtle drawing that shows the following.

Check each item as you add it to your code — checked items fade and strike through so you can see what is left.

When every box is checked, upload your code and screenshot to the class shared folder.

This should feel easy, creative, and fun. The goal is not perfection — make something appear, change it, and see how your code controls the art.

🕹️ What to do
  1. Run the starter example below.
  2. Change at least 3 colors.
  3. Change at least 2 sizes.
  4. Change at least 1 x/y location.
  5. Add comments that explain your design.
  6. Take a screenshot when your drawing works.
  7. Upload your .py file and your screenshot to the class shared folder.

🧪 Open your editor and write your own code from scratch, or use Mr. Brett's template and modify it. Try the Python Sandbox turtle IDE, or Trinket, Replit, or IDLE.
