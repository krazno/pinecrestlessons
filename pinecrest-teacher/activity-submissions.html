Review student activity turn-ins

Students turn in work on the student lesson page (name, level, screenshot, and code) in the section right above the exit ticket.

Enable online review (recommended)

  1. Create a free form at formspree.io with file uploads enabled.
  2. Copy your form id (the part after /f/ in the form URL).
  3. In pinecrest-student/index.html, set ACTIVITY_SUBMIT.formspreeId to that id.
  4. Redeploy the site. Submissions appear in your Formspree inbox with student name, level, screenshot, and code.

Until formspreeId is set, turn-ins are saved only on each student's browser (they still see their history, but you cannot review them remotely).

What students submit

  • Level 1, 2, or 3 — three placeholder activity cards on the page today
  • Screenshot — image of their work
  • Code — .py file or pasted Python
