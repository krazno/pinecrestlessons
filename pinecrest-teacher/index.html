✨The Full Prompt Glow-UpAll five moves combined
❌ Weak

Make a review activity for my students.

✅ Improved

Act as an experienced Pine Crest teacher. Before answering, ask me 3 clarifying questions about my class, topic, and goal. Then create a 10-minute review activity using this format: 1. Warm-up 2. Partner task 3. Exit ticket First identify the skill the activity builds and explain why the activity supports that skill. Then roleplay as a student who might be confused and ask me one question.
